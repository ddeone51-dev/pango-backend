## ğŸ†˜ Troubleshooting

### **Can't Login?**
```
1. Check email: admin@pango.co.tz
2. Check password (caps lock on?)
3. Clear browser cache
4. Try incognito/private window
5. Check backend is running
```

### **Dashboard Not Loading?**
```
1. Refresh page (Ctrl+R)
2. Check network connection
3. Try different browser
4. Check browser console (F12)
5. Restart backend server
```

### **Can't Find User/Property?**
```
1. Use search box
2. Check filters (might be hidden)
3. Try different search terms
4. Check pagination (next page?)
5. Scroll down if table is long
```

### **Export Not Working?**
```
1. Check browser allows downloads
2. Try different file format (CSV vs PDF)
3. Check browser console for errors
4. Disable ad blockers
5. Try different browser
```
